# Roadmap

## Phase 1 - Foundation
- Initialize app, database, auth provider, storage, and baseline UI shell.

## Phase 2 - Ingestion + Parsing
- Resume upload, extraction, LLM call, schema validation, and error handling.

## Phase 3 - Portfolio Builder
- Form editing, link management, hero image upload, theme selection, live preview.

## Phase 4 - Publishing
- Slug routing, public portfolio rendering, and SEO basics.

## Phase 5 - Verification + Deployment
- Run tests, manual QA checklist, deploy to Vercel, and verify production URL.
